Inline Elements

  1. Inline element odha default behaviour ena na adhu odha content width mathum tha edhuthukom.
  2. So, remaining irukura space next element occupy panikum adhuvum oru inline element ah irunda(it never start in a new line).
  3. Adhae matri oru inline element ku height and width apply agadhu reason adhoda content width tha edhuthukom so no use of using height and width property.
  4. If you want to use height and width property in inline adhoda display property change pananum.
  5. We cannot add margin top and bottom to an inline element.

Block Level Elements

  1. Block level elements adhodha parent element odha full width edhuthukum. So, it start on an new line.
  2. Block level element ku height and width property apply panalam because it takes full width of the parent element.
  3. By default block level element margin top and bottom irukum nambo use pandra block element poruthu margin top and bottom value apply agum.

Inline Block

  1. Inline block epo use agum na,oru element inline agavum irukanum adhae samayathula height and width apply aganum na apo inline element odha display property inline block nu change pananum.
  2. Inline block nu display panum podu anda element inline ah irukum but block level act ah agum.
  3. It never start on a new line.
